Zona
—

El marcador del mapa es arrastrable: si el geocoding no cayó exactamente sobre el lote, movelo a mano y la zona se recalcula sola.

Cómo funciona / limitaciones
  • El geocoding usa Nominatim (OpenStreetMap), gratuito y sin API key. No es tan preciso como Google Maps en loteos nuevos o calles sin nombre oficial todavía — por eso el marcador se puede arrastrar.
  • La capa de zonificación se procesa entera en el navegador (no se sube a ningún servidor).
  • Si el punto no cae dentro de ningún polígono, es probable que el marcador esté justo en el borde o la dirección no haya geocodificado bien — arrastralo un poco.
  • Para uso real, conviene descargar la capa una sola vez y dejarla ya cargada en el sistema (esto es un prototipo de validación del flujo).
